# Python 分布式区块链系统 (v2)

这是一个使用 Python 从零构建的简易分布式区块链和交易系统。
**v2 版本更新：**
1.  **交易签名**: 使用 ECDSA (公私钥) 确保交易的真实性。
2.  **P2P Gossip**: 节点注册时会自动发现网络中的其他节点。

## 功能特性

* **基础区块链**: 区块结构、哈希链接、创世区块。
* **交易系统**: 支持创建和存储交易数据。
* **工作量证明 (PoW)**: 简单的挖矿算法来保障网络安全。
* **分布式共识**: 实现“最长链原则”解决节点间的数据冲突。
* **API 接口**: 提供 RESTful API 进行交互（交易、挖矿、查询）。
* **数据持久化**: 区块链数据自动保存到本地 JSON 文件。
* **钱包系统 (ECDSA)**: 使用公私钥对交易进行签名和验证。
* **Gossip 协议**: 通过 `/nodes/register` 接口实现简单的节点自动发现。
